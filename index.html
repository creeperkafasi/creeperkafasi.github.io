-XdotoolJS-

XdotoolJS is a simple deno.js wrapper of jordansissel's X11 automation program, xdotool.

You can check out the repository on GitHub
Or you can get the package from deno.land

Just paste;

import xdotool from "https://deno.land/x/xdotool/mod.ts";



to import it into your deno.js project.
